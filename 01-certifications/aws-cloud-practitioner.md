# AWS Cloud Practitioner Essentials

## Status
In progress

## Why this course
To build foundational understanding of cloud concepts, AWS core services, and the shared responsibility model.

## Key concepts learned so far
- What cloud computing is and why organisations adopt it
- AWS shared responsibility model
- Difference between IaaS, PaaS, and SaaS
- Core AWS services and their purpose

## Security perspective
Even at the foundational level, this course highlights how security is shared between the cloud provider and the customer, and why identity, logging, and configuration matter early.
